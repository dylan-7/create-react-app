# joys-react-h5-scripts

```
 create-react-app --scripts-version=joys-react-h5-scripts my-app
```

## 目录结构

config 配置

build 打包

public 同步打包好的代码，用于服务器

src 开发

typings typescript类型文件

view html 模板

[D] Deprecated #1138: 一项功能过时/弃用
